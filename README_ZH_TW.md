## 給開發者的資訊

### 開發環境建置

- Node.js 版本最低需求: 10.19.0以上

**使用 yarn 的做法**
``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev
```

**使用 docker compose 的做法**
```bash
# serve with hot reload at localhost:3000
$ docker-compose up --build
```
